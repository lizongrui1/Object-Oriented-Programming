* 结构体与继承
1. 什么是结构体
是将多个任意类型的变量组合在一起的聚合数据类型
可以理解为Go语言的结构体struct和其他语言的class有相等的地位，但是Go语言放弃大量面向对象的特性，所有的Go语言类型除了指针类型外，都可以有自己的方法，提高了可扩展性。在 Go 语言中没有没有 class 类的概念，只有 struct 结构体的概念，因此也没有继承。

2. 定义方法

}`
